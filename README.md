# Angular PrimeNG Sandbox

A learning project for exploring and experimenting with **Angular 21**, **PrimeNG**, **Tailwind CSS**, and the latest web framework features.

## 🎯 Purpose

This project serves as a sandbox for:

- Learning Angular 21+ features and best practices
- Experimenting with PrimeNG UI components
- Exploring Tailwind CSS integration
- Testing modern web development patterns
- Understanding Angular's latest updates

## 🚀 Tech Stack

- **Angular**: 21.2.11 - Modern JavaScript framework
- **PrimeNG**: 21.1.7 - Professional UI component library
- **Tailwind CSS**: 4.3.0 - Utility-first CSS framework
- **TypeScript**: 5.9 - Strongly typed JavaScript
- **RxJS**: 7.8 - Reactive programming library
- **Node.js**: 20+ - JavaScript runtime

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/angular-primeng-sandbox.git
cd angular-primeng-sandbox

# Install dependencies
npm install
```

## 🏃 Development Server

Start the development server:

```bash
npm start
```

Navigate to `http://localhost:4200/` in your browser. The application will automatically reload when you modify source files.

## 🏗️ Project Structure

```
src/
├── app/
│   ├── app.ts              # Root component
│   ├── app.html            # Root template
│   ├── app.css             # Component styles
│   ├── app.routes.ts       # Route configuration
│   └── app.config.ts       # Application configuration
├── main.ts                 # Application entry point
├── styles.css              # Global styles (includes Tailwind)
└── index.html              # HTML entry point
```

## 🛠️ Available Scripts

- `npm start` - Start development server
- `npm run build` - Build for production
- `npm run watch` - Watch mode for development
- `npm test` - Run unit tests
